## Setup

1. Install deps

```
# `preinstall` will install hexo globally
# and config my local git user
# you **SHOULD FIX** this if you want make you own
npm install && npm run preinstall
```

2. Run server

```
# This step
hexo server
```

3. Write post

```
hexo new post "your-post-name"
```
