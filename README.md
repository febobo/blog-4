## Setup

1. Install deps

```
npm install

# npm config user.email 'pengfei_yang@outlook.com'
```

2. Run server

```
# This step is optional
hexo server
```

3. Write post

```
hexo new post "your-post-name"
```

4. Deploy to github

```
npm run deploy
```
